Comandos para fusionar ramas en Git 🔄

📜 Comandos Útiles para Fusionado de Ramas

  • git merge nombrarama: 🔗 Une los cambios de nombrarama con la rama actual (se debe realizar desde la rama principal). Puedes incluir un "mensaje opcional" para describir el merge.
  • git merge --abort: ⏸️ Anula el proceso de merge y restaura el estado anterior, deshaciendo cualquier cambio realizado durante el merge.

⚠️ Resolución de conflictos al hacer un merge

Cuando unificas cambios, puede haber conflictos. Si esto ocurre, deberás decidir cuál parte del código conservar y cuál descartar. Asegúrate de revisar cada conflicto cuidadosamente antes de finalizar el merge.
