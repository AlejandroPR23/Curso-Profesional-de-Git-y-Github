Enviar Diferentes Branches a GitHub 🌿

🌿 Mostrar la Lista de Branches Actuales

  • Para ver la lista de branches que existen en tu repositorio, ejecuta:
  • git show-branch

🔍 Mostrar Lista de Branches con Más Detalle

  • Para obtener una vista más detallada de todos los branches, utiliza:
  • git show-branch --all

🖥️ Abrir Aplicación de Visualización de Branches y Commits

  • Para abrir una aplicación que permite ver branches y commits, usa:
  • gitk

📤 Enviar una Rama a GitHub

  • Para enviar la rama actual a GitHub, ejecuta:
  • git push origin nombrebranch

🚀 Subir Commits y la Rama Estableciendo el Upstream

  • Para subir los commits de una rama y establecer el upstream:
  • git push -u origin nombrebranch

➕ Crear Nuevos Branches

  • Para crear un nuevo branch llamado header, ejecuta:
  • git branch header
  • Para crear otro branch llamado footer, usa:
  • git branch footer

⬇️ Traer una Rama Específica del Repositorio

  • Para obtener una rama específica del repositorio, ejecuta:
  • git pull origin nombrerama
