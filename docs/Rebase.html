Rebase en Git ⚙️

Nota Importante

El uso de rebase en repositorios remotos es considerado una mala práctica. Se recomienda realizarlo únicamente en repositorios locales.

Pasos para Realizar un Rebase

1. Crea una Nueva Rama:

Ejecuta el siguiente comando:

git checkout -b nueva-rama

2. Realiza los Cambios:

Haz los cambios que desees en esa rama.

3. Ejecuta el Rebase:

Estando en la rama creada, ejecuta:

git rebase main

4. Resolver Conflictos:

Si hay conflictos, resuélvelos y continúa con el siguiente comando:

git rebase --continue

5. Cambia a la Rama Principal:

Ejecuta el siguiente comando:

git checkout main

6. Fusiona la Rama Rebasa con la Principal:

Finalmente, ejecuta:

git rebase nueva-rama

Ahora las dos ramas se habrán fusionado correctamente.
