Comandos de Terminal (Git Bash)

Una guía práctica para comandos de terminal utilizados en Git Bash.

Navegación y Gestión de Directorios

  • pwd - Muestra en qué carpeta estás actualmente.
  • cd <directorio> - Cambia de directorio.
  • cd .. - Vuelve al directorio anterior.

Listado de Archivos

  • ls - Lista los archivos del directorio actual.
  • ls -al - Lista detallada de todos los archivos, incluidos los ocultos.
  • ls -l - Lista detallada de los archivos visibles.
  • ls -a - Muestra todos los archivos, incluidos los ocultos.

Comandos de Gestión de Archivos

  • clear - Limpia la consola.
  • mkdir <nombre_carpeta> - Crea un nuevo directorio.
  • touch <archivo> - Crea un archivo vacío.
  • nano <archivo> - Abre un editor de texto en la terminal.
  • cat <archivo> - Muestra el contenido de un archivo.
  • rm <archivo> - Elimina un archivo.

Historial y Ayuda

  • history - Muestra el historial de comandos usados.
  • <comando> --help - Muestra información sobre un comando.
