
Comandos terminal (GitBash) muchos no funcionan en powersheel(terminal windows)

pwd

Te muestra en que carpeta estas ahora mismo

cd 

Permite cambiar de directorio

cd ..

salir del directorio actual

ls 

Muestra la lista de archivos del directorio actual

ls -al

Muestra todos los archivos del directorio incluso los ocultos y los muestra en una lista

ls -l

muestra los archivos del directorio que no estan ocultos

ls -a  

muestra el grupo de archivos del directorio pero no en una lista

clear   (ctrl+L)

limpiar la consola

mkdir nombrecarpeta

Crea un directorio

touch vacio.txt

nano text.txt (Abre una terminal de GNU donde directamente se puede escribir el interior del archivo en caso no exista lo crea)

permite crear un archivo vacio

cat vacio.txt

permite ver el contenido de un archivo en la consola

history

Muestra la lista de comandos escrita en la terminal durante el tiempo que se uso

se puede utilizar un comando pasado solo usando el numero que aparece en la lista con otro comando:

!7

rm vacio.txt

Sirve para eliminar un archivo


comando --help

Explica los comandos
